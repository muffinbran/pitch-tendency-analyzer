# Pitch Tendency Analyzer

A modern, web app designed to help musicians understand their intonation.

## Features

### Current Features

- Audio capture in browser
- Real-time tuner and pitch detection
- Creation of different profiles for instruments
- Table view of instrument pitch tendencies over time

### Upcoming Features

- Graph analysis of pitch tendencies
- Exportable reports

## Tech Stack

### Frontend

- React (Vite)
- TypeScript
- TailwindCSS
- Web Audio API

### Backend

- FastAPI
- Python
- SQLAlchemy
- pydantic

### Dev Tools

- Prettier/Black (formatter)
- uvicorn (live reload)
